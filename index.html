  • Default position is set to static
    • The element sits in its normal place in the document
  • Set to relative
    • The element still takes up space in the document, but now the top and left style properties can be used to move it realtive to its normal place
  • Set to absolute
    • The element is removed from the normal document flow, that is, it no longer takes up space and may overlap with other elements
